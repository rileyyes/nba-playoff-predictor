# 🏀 NBA Playoff Simulator

Machine learning powered NBA playoff simulator built with Python and Streamlit.

## Features

- Select any 16-team playoff bracket
- Eastern and Western Conference validation
- Best-of-7 series simulation
- Full playoff bracket generation
- Conference Finals and NBA Finals
- Team logos and playoff visualization
- Machine learning based win predictions

## Technologies

- Python
- Streamlit
- Pandas
- Scikit-Learn
- NBA API
- Joblib

## Run

```bash
streamlit run app/streamlit_app.py
